# 🏆 Gestor de Álbum Estampas Mundial 2026 - Panini

## 📋 Descripción General

Página web interactiva y profesional para gestionar y controlar tu colección de estampas del **Álbum Panini del Mundial 2026**. Incluye todas las características solicitadas con una interfaz moderna, responsiva y totalmente funcional.

---

## ✨ Características Principales

### 1. **Sección Grupos Mundialistas** ⚽
- Organización exacta de los **12 grupos oficiales** del Mundial 2026 (A, B, C... L)
- **4 equipos por grupo** según sorteo oficial
- **13 estampas por equipo** (aproximadamente)
- Tabla interactiva con:
  - Número de estampa
  - Nombre del jugador
  - Control de "¿Tengo?" mediante checkboxes
- Contador automático por grupo que muestra progreso en tiempo real
- Colores distintivos para cada grupo

### 2. **Sección Equipos Mundialistas** 🌍
- Vista completa de los **48 equipos participantes**
- Tabla con columnas:
  - Equipo
  - Total de estampas
  - Estampas obtenidas
  - Estampas faltantes
- Actualización automática según cambios en grupos
- Búsqueda y filtrado de equipos

### 3. **Sección Elementos del Mundial** ⚽
- Tabla de elementos especiales:
  - **Balón Oficial Trionda** (10 estampas)
  - **Copa del Mundo** (8 estampas)
  - **Mascotas** (Maple, Zayu, Clutch) - 15 estampas
  - **Estadios** (16 estampas)
  - **Insignias de Naciones** (48 estampas)
  - **Otros Elementos Especiales** (12 estampas)
- Contador progresivo con barra de progreso
- Control individual para cada elemento

### 4. **Sección Coca-Cola** 🥤
- **12 estampas especiales** de la colección Coca-Cola
- Tabla dedicada con:
  - Número de estampa (COCA-1, COCA-2, etc.)
  - Descripción de selecciones
  - Control de "¿Tengo?"
- Contador total de Coca-Cola

### 5. **Sección Repetidas** 📋
- **Selector desplegable dinámico** para elegir estampa del catálogo completo
- Seguimiento de cantidad de repetidas
- Cálculo automático de repetidas totales
- Tabla interactiva con opción de eliminar registros

### 6. **Panel de Control Visual** 📊
- **Métricas destacadas en tarjetas:**
  - Total de estampas obtenidas
  - Total de estampas faltantes
  - Porcentaje de completitud
  - Cantidad de estampas repetidas

- **Gráficos interactivos:**
  - Gráfico de progreso por sección (Doughnut)
  - Gráfico de progreso por grupo (Barras)
  - Gráfico de línea con evolución

- **Desglose por sección:**
  - Grupos
  - Elementos
  - Coca-Cola
  - Repetidas

## 🚀 Cómo Usar

### Requisitos
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (solo para cargar Chart.js)

### Pasos para Usar

1. **Descargar el archivo:** `album-mundial-2026.html`

2. **Abrir en navegador:**
   - Hacer doble clic en el archivo
   - O arrastrarlo a la ventana del navegador

3. **Explorar secciones:**
   - Clic en botones de navegación superior
   - Marcar/desmarcar estampas según las tengas

4. **Ver progreso:**
   - Panel de Control muestra resumen en tiempo real
   - Gráficos actualizan automáticamente

5. **Guardar progreso:**
   - Automático constantemente
   - Opcional: descargar como JSON

## 📝 Instrucciones Detalladas

### Agregar una Estampa
1. Navega a "Grupos Mundialistas"
2. Encuentra el grupo y equipo
3. Marca el checkbox junto a la estampa

### Ver Progreso General
1. En "Panel de Control"
2. Observa las tarjetas con métricas
3. Revisa los gráficos de progreso

### Registrar Repetidas
1. Ve a "Repetidas"
2. Haz clic en "+ Agregar Repetida"
3. Ingresa el número de estampa y cantidad
4. Confirma

### Descargar Progreso
1. En "Panel de Control"
2. Clic en "📥 Descargar Progreso"
3. Se descargará un JSON con tus datos
4. Guárdalo en un lugar seguro

### Cargar Progreso
1. En "Panel de Control"
2. Clic en "📤 Cargar Progreso"
3. Selecciona tu archivo JSON guardado
4. Tus datos se restaurarán
